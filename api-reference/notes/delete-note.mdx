# Delete Note

Delete a note.

## Request

```bash
DELETE /api/external/v1/notes/:id
```

## Response

```json
{
  "message": "Note deleted successfully"
}
```

## Error Responses

| Status Code | Description                    | Example                                          |
| ----------- | ------------------------------ | ------------------------------------------------ |
| 401         | Invalid or missing API key     | `{"error": "Could not authenticate, invalid API Key"}` |
| 404         | Note not found                 | `{"error": "Note not found"}`                    |
