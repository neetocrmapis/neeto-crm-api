# Get Company

Retrieve a single company by ID.

## Request

```bash
GET /api/external/v1/companies/:id
```

## Response

```json
{
  "company": {
    "id": "1",
    "name": "Acme Inc",
    "address": "123 Main St",
    "organization_id": "1",
    "created_at": "2024-01-01T00:00:00Z",
    "updated_at": "2024-01-01T00:00:00Z",
    "tag_ids": [1, 2],
    "field_values": [
      {
        "id": "1",
        "field_id": "1",
        "value": "custom value"
      }
    ]
  }
}
```

## Error Responses

| Status Code | Description                    | Example                                          |
| ----------- | ------------------------------ | ------------------------------------------------ |
| 401         | Invalid or missing API key     | `{"error": "Could not authenticate, invalid API Key"}` |
| 404         | Company not found              | `{"error": "Company not found"}`                 |
