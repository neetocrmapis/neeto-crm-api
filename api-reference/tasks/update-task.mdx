# Update Task

Update an existing task.

## Request

```bash
PATCH /api/external/v1/tasks/:id
```

## Request Body

| Parameter   | Type    | Description      |
| ----------- | ------- | ---------------- |
| title       | string  | Task title       |
| description | string  | Task description |
| status      | string  | Task status      |
| priority    | string  | Task priority    |
| due_date    | date    | Due date         |
| due_time    | time    | Due time         |
| owner_id    | integer | Owner user ID    |

## Example

```json
{
  "task": {
    "title": "Updated task title",
    "status": "completed",
    "priority": "low"
  }
}
```

## Response

```json
{
  "task": {
    "id": "1",
    "title": "Updated task title",
    "status": "completed",
    "priority": "low",
    "organization_id": "1",
    "created_at": "2024-01-01T00:00:00Z",
    "updated_at": "2024-01-02T00:00:00Z"
  }
}
```

## Error Responses

| Status Code | Description                    | Example                                          |
| ----------- | ------------------------------ | ------------------------------------------------ |
| 401         | Invalid or missing API key     | `{"error": "Could not authenticate, invalid API Key"}` |
| 404         | Task not found                 | `{"error": "Task not found"}`                    |
| 422         | Validation error               | `{"errors": ["Title can't be blank"]}`           |
