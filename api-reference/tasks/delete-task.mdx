# Delete Task

Delete a task from your CRM.

## Request

```bash
DELETE /api/external/v1/tasks/:id
```

## Response

```json
{
  "message": "Task deleted successfully"
}
```

## Error Responses

| Status Code | Description                    | Example                                          |
| ----------- | ------------------------------ | ------------------------------------------------ |
| 401         | Invalid or missing API key     | `{"error": "Could not authenticate, invalid API Key"}` |
| 404         | Task not found                 | `{"error": "Task not found"}`                    |
