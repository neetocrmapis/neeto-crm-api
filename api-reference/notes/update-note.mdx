# Update Note

Update an existing note.

## Request

```bash
PATCH /api/external/v1/notes/:id
```

## Request Body

| Parameter     | Type   | Description          |
| ------------- | ------ | -------------------- |
| note          | object | Note object          |
| note[content] | string | Updated note content |

## Example

```json
{
  "note": {
    "content": "Updated note content"
  }
}
```

## Response

```json
{
  "note": {
    "id": "1",
    "content": "Updated note content",
    "created_at": "2024-01-01T00:00:00Z",
    "updated_at": "2024-01-02T00:00:00Z"
  }
}
```

## Error Responses

| Status Code | Description                    | Example                                          |
| ----------- | ------------------------------ | ------------------------------------------------ |
| 401         | Invalid or missing API key     | `{"error": "Could not authenticate, invalid API Key"}` |
| 404         | Note not found                 | `{"error": "Note not found"}`                    |
| 422         | Validation error               | `{"errors": ["Content can't be blank"]}`         |
