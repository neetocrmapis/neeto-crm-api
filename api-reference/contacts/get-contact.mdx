# Get Contact

Retrieve a single contact by ID.

## Request

```bash
GET /api/external/v1/contacts/:id
```

## Response

```json
{
  "contact": {
    "id": "1",
    "name": "John Doe",
    "email": "john@example.com",
    "phone_number": "+1234567890",
    "company_id": "1",
    "organization_id": "1",
    "created_at": "2024-01-01T00:00:00Z",
    "updated_at": "2024-01-01T00:00:00Z",
    "tag_ids": [1, 2],
    "owner": {
      "id": "1",
      "name": "Jane Smith",
      "email": "jane@example.com"
    },
    "company": {
      "id": "1",
      "name": "Acme Inc"
    },
    "field_values": [
      {
        "id": "1",
        "field_id": "1",
        "value": "custom value"
      }
    ]
  }
}
```

## Error Responses

| Status Code | Description                    | Example                                          |
| ----------- | ------------------------------ | ------------------------------------------------ |
| 401         | Invalid or missing API key     | `{"error": "Could not authenticate, invalid API Key"}` |
| 404         | Contact not found              | `{"error": "Contact not found"}`                 |
