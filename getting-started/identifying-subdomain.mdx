# Identifying Your Subdomain

Your NeetoCRM workspace is identified by a unique subdomain. This subdomain is used in the API base URL.

## Finding Your Subdomain

Your subdomain is the part of the URL you use to access your NeetoCRM workspace:

```
https://your-workspace.neetocrm.com
```

In this example, `your-workspace` is your subdomain.

## API Base URL

All API requests should be made to:

```
https://your-workspace.neetocrm.com/api/external/v1
```

Replace `your-workspace` with your actual workspace subdomain.

## Example

If your workspace is at `acme.neetocrm.com`, your API requests would go to:

```
https://acme.neetocrm.com/api/external/v1
```
