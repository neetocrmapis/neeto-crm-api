## Step 1: Make Your First API Call

Let's retrieve all contacts in your workspace using a [GET request](https://en.wikipedia.org/wiki/HTTP#Request_methods):

<CodeGroup>
```bash curl
curl -X GET "https://{your-subdomain}.neetocrm.com/api/external/v1/contacts" \
  -H "X-Api-Key: your-api-key" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch(
  "https://{your-subdomain}.neetocrm.com/api/external/v1/contacts",
  {
    method: "GET",
    headers: {
      "X-Api-Key": "your-api-key",
      "Content-Type": "application/json",
    },
  }
);

const data = await response.json();
console.log(data);
```

```python Python
import requests

url = "https://{your-subdomain}.neetocrm.com/api/external/v1/contacts"
headers = {
    "X-Api-Key": "your-api-key",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
data = response.json()
print(data)
```

</CodeGroup>

<Warning>
  Replace `your-subdomain` with your actual workspace subdomain and
  `your-api-key` with your generated API key.
</Warning>

## Step 2: Understanding the Response

A successful response will look like this [JSON](https://en.wikipedia.org/wiki/JSON) format:

```json
{
  "contacts": [
    {
      "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
      "name": "John Doe",
      "email": "john@example.com",
      "phone_number": "+1234567890",
      "owner": null,
      "tags": [],
      "company": {
        "id": null,
        "name": null
      },
      "created_at": "2024-01-01T00:00:00Z",
      "updated_at": "2024-01-01T00:00:00Z"
    }
  ],
  "pagination": {
    "total_records": 1,
    "total_pages": 1,
    "current_page_number": 1,
    "page_size": 30
  }
}
```
