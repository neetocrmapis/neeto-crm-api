# Create Company

Create a new company in your CRM.

## Request

```bash
POST /api/external/v1/companies
```

## Request Body

| Parameter | Type   | Required | Description      |
| --------- | ------ | -------- | ---------------- |
| name      | string | Yes      | Company name     |
| address   | string | No       | Company address  |
| tag_ids   | array  | No       | Array of tag IDs |

## Example

```json
{
  "company": {
    "name": "Acme Inc",
    "address": "123 Main St, New York, NY"
  }
}
```

## Response

```json
{
  "company": {
    "id": "1",
    "name": "Acme Inc",
    "address": "123 Main St, New York, NY",
    "organization_id": "1",
    "created_at": "2024-01-01T00:00:00Z",
    "updated_at": "2024-01-01T00:00:00Z"
  }
}
```

## Error Responses

| Status Code | Description                    | Example                                          |
| ----------- | ------------------------------ | ------------------------------------------------ |
| 401         | Invalid or missing API key     | `{"error": "Could not authenticate, invalid API Key"}` |
| 422         | Validation error               | `{"errors": ["Name can't be blank"]}`            |
