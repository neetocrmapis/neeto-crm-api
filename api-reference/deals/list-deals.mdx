# List Deals

Retrieve a paginated list of deals in your CRM.

## Request

```bash
GET /api/external/v1/deals
```

## Query Parameters

| Parameter   | Type    | Description                    |
| ----------- | ------- | ------------------------------ |
| page_number | integer | Page number (default: 1)       |
| page_size   | integer | Records per page (default: 30) |
