# Create Deal

Create a new deal in your CRM.

## Request

```bash
POST /api/external/v1/deals
```

## Request Body

| Parameter   | Type    | Required | Description                  |
| ----------- | ------- | -------- | ---------------------------- |
| name        | string  | Yes      | Deal name                    |
| value       | decimal | No       | Deal value                   |
| currency    | string  | No       | Currency code (default: USD) |
| status      | string  | No       | Deal status                  |
| company_id  | integer | No       | Associated company ID        |
| contact_id  | integer | No       | Associated contact ID        |
| owner_id    | integer | No       | Owner user ID                |
| stage_id    | integer | No       | Stage ID                     |
| pipeline_id | integer | No       | Pipeline ID                  |
| tag_ids     | array   | No       | Array of tag IDs             |

## Example

```json
{
  "deal": {
    "name": "Big Deal",

    "currency": "USD",
    "company_id": 1,
    "contact_id": 1,
    "stage_id": 1,
    "pipeline_id": 1
  }
}
```
