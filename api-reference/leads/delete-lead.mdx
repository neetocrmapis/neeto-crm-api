# Delete Lead

Delete a lead from your CRM.

## Request

```bash
DELETE /api/external/v1/leads/:id
```

## Response

```json
{
  "message": "Lead deleted successfully"
}
```

## Error Responses

| Status Code | Description                    | Example                                          |
| ----------- | ------------------------------ | ------------------------------------------------ |
| 401         | Invalid or missing API key     | `{"error": "Could not authenticate, invalid API Key"}` |
| 404         | Lead not found                 | `{"error": "Lead not found"}`                    |
