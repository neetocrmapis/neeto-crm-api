# Delete Company

Delete a company from your CRM.

## Request

```bash
DELETE /api/external/v1/companies/:id
```

## Response

```json
{
  "message": "Company deleted successfully"
}
```

## Error Responses

| Status Code | Description                    | Example                                          |
| ----------- | ------------------------------ | ------------------------------------------------ |
| 401         | Invalid or missing API key     | `{"error": "Could not authenticate, invalid API Key"}` |
| 404         | Company not found              | `{"error": "Company not found"}`                 |
