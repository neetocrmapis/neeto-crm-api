# Update Lead

Update an existing lead.

## Request

```bash
PATCH /api/external/v1/leads/:id
```

## Request Body

| Parameter  | Type    | Description           |
| ---------- | ------- | --------------------- |
| name       | string  | Lead name             |
| status     | string  | Lead status           |
| company_id | integer | Associated company ID |
| contact_id | integer | Associated contact ID |
| owner_id   | integer | Owner user ID         |

## Example

```json
{
  "lead": {
    "name": "John Smith",
    "status": "qualified"
  }
}
```

## Response

```json
{
  "lead": {
    "id": "1",
    "name": "John Smith",
    "status": "qualified",
    "company_id": "1",
    "owner_id": "1",
    "organization_id": "1",
    "created_at": "2024-01-01T00:00:00Z",
    "updated_at": "2024-01-02T00:00:00Z"
  }
}
```
