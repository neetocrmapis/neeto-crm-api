# Delete Deal

Delete a deal from your CRM.

## Request

```bash
DELETE /api/external/v1/deals/:id
```

## Response

```json
{
  "message": "Deal deleted successfully"
}
```

## Error Responses

| Status Code | Description                    | Example                                          |
| ----------- | ------------------------------ | ------------------------------------------------ |
| 401         | Invalid or missing API key     | `{"error": "Could not authenticate, invalid API Key"}` |
| 404         | Deal not found                 | `{"error": "Deal not found"}`                    |
