# Create Task

Create a new task in your CRM.

## Request

```bash
POST /api/external/v1/tasks
```

## Request Body

| Parameter   | Type    | Required | Description           |
| ----------- | ------- | -------- | --------------------- |
| title       | string  | Yes      | Task title            |
| description | string  | No       | Task description      |
| status      | string  | No       | Task status           |
| priority    | string  | No       | Task priority         |
| due_date    | date    | No       | Due date              |
| due_time    | time    | No       | Due time              |
| owner_id    | integer | No       | Owner user ID         |
| contact_id  | integer | No       | Associated contact ID |
| company_id  | integer | No       | Associated company ID |
| deal_id     | integer | No       | Associated deal ID    |
| lead_id     | integer | No       | Associated lead ID    |

## Example

```json
{
  "task": {
    "title": "Follow up with client",
    "description": "Send proposal",
    "status": "pending",
    "priority": "high",
    "due_date": "2024-01-15",
    "owner_id": 1,
    "contact_id": 1
  }
}
```
