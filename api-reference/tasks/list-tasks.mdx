# List Tasks

Retrieve a paginated list of tasks in your CRM.

## Request

```bash
GET /api/external/v1/tasks
```

## Query Parameters

| Parameter   | Type    | Description                    |
| ----------- | ------- | ------------------------------ |
| page_number | integer | Page number (default: 1)       |
| page_size   | integer | Records per page (default: 30) |
